This guide takes you from zero to writing code with Loophole. No prior setup needed.

## Before you start

Make sure you have:

- A computer running macOS, Windows, or Linux
- About 5 minutes
- A project you want to work on (or you can make a new one)

## Steps

<Steps>
  <Step title="Download Loophole">
    Head to our [GitHub releases page](https://github.com/loophole-ai/loophole-ide/releases) and grab the latest version for your operating system.

    Or check the [Installation guide](/getting-started/installation) for step-by-step instructions per platform.
  </Step>
  <Step title="Open it up">
    Install Loophole like any normal app. On macOS, drag it to Applications. On Windows, run the installer. On Linux, run the AppImage or `.deb` package.

    Open Loophole when it's done.
  </Step>
  <Step title="Open a project">
    Click **File → Open Folder** and pick any folder with code in it. If you don't have one, make a new folder and open that.
  </Step>
  <Step title="Try the AI chat">
    Press `Cmd+L` (Mac) or `Ctrl+L` (Windows/Linux) to open the chat panel.

    Try asking something like:

    ```text
    What does this project do?
    ```

    Loophole will read your project and answer.
  </Step>
  <Step title="Make your first edit">
    Highlight a piece of code, press `Cmd+K` (Mac) or `Ctrl+K` (Windows/Linux), and describe what you want changed. For example:

    ```text
    Add error handling to this function
    ```

    Loophole rewrites the code right there.
  </Step>
</Steps>
