The chat panel is where you talk to Loophole. Ask questions, get explanations, request code — anything goes.

## Open the chat

Press `Cmd+L` (Mac) or `Ctrl+L` (Windows/Linux). The chat panel slides in from the right.

## What you can ask

Pretty much anything. Some examples:

- _"What does this function do?"_
- _"Why is this test failing?"_
- _"Add a new route to the Express app that returns the current time."_
- _"Convert this React class component to a function component."_
- _"Write tests for the user service."_

## Add context with @

Type `@` in the chat box to pull stuff into the conversation:

- `@filename` — add a specific file
- `@folder` — add a whole folder
- `@codebase` — let Loophole search everything
- `@docs` — search official docs for libraries you use
- `@web` — search the internet

The more context you give it, the better its answers.

## Switch models

Click the model name at the bottom of the chat to switch between GPT, Claude, Gemini, or your local model. Different models are good at different things.

## Chat history

All your conversations are saved. Click the history icon at the top of the chat to find old ones.

## Tips

- Keep your messages clear and specific
- If the AI gives a bad answer, tell it why — it'll try again
- Use `@codebase` when you're not sure where to point it
