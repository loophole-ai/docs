## Switch themes

Open the command palette (`Cmd+Shift+P` / `Ctrl+Shift+P`) and type **Color Theme**. Pick one from the list.

Loophole ships with a few themes out of the box:

- Loophole Dark (default)
- Loophole Light
- High Contrast
- Solarized

Want more? Install a theme extension from the [extensions panel](/guides/extensions).

## Match your system

Go to **Settings → Appearance** and turn on **Sync with system theme**. Loophole will switch between light and dark when your OS does.

## Font

In **Settings → Editor**, you can change:

- Font family (any font on your system works)
- Font size
- Line height
- Whether to use font ligatures

Good coding fonts to try: **JetBrains Mono**, **Fira Code**, **Cascadia Code**, **Monaspace**.

## Density

If the UI feels too packed or too spacious, change **Settings → Appearance → Density** between Compact, Default, and Roomy.

## Zen mode

Need to focus? Press `Cmd+K Z` (Mac) or `Ctrl+K Z` (Windows/Linux) to hide everything except your code. Press `Esc Esc` to come back.
