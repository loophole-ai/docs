Loophole supports extensions, so you can add the languages, themes, and tools you want.

## Install one

<Steps>
  <Step title="Open the extensions panel">
    Click the puzzle piece icon on the left sidebar, or press `Cmd+Shift+X` / `Ctrl+Shift+X`.
  </Step>
  <Step title="Search">
    Type the name of what you want, like `Python` or `Dracula theme`.
  </Step>
  <Step title="Install">
    Click **Install** next to the one you want.
  </Step>
</Steps>

## VS Code extensions

Most VS Code extensions work in Loophole. If you can't find one in our marketplace, you can install it as a `.vsix` file:

1. Download the `.vsix` from the VS Code Marketplace
2. Open the Loophole command palette (`Cmd+Shift+P` / `Ctrl+Shift+P`)
3. Type **Install from VSIX** and pick the file

## Recommended extensions

Here are some that pair well with Loophole:

- **Prettier** — auto-format your code
- **GitLens** — fancier Git tools
- **Error Lens** — see errors inline
- **Path Intellisense** — autocomplete file paths

## Manage what's installed

Go to the extensions panel and click **Installed** to see everything. From there you can disable, uninstall, or update extensions.
