## Anthropic

1. Go to [console.anthropic.com](https://console.anthropic.com) and sign in.
2. Navigate to **API Keys** → **Create Key**.
3. Name the key (e.g. "Loophole") and copy it.
4. Add a payment method under **Billing** — Anthropic requires a card even if you're testing.

<Tip>
  Start with a low spending limit (e.g. \$10/month) in **Billing → Usage Limits** to avoid surprise charges while experimenting.
</Tip>

---

## OpenAI

1. Go to [platform.openai.com](https://platform.openai.com) and sign in.
2. Navigate to **API Keys** → **Create new secret key**.
3. Name it and copy it immediately — you won't see it again.
4. New accounts get \$5 in free credits. Add a payment method under **Billing** once exhausted.

---

## Google Gemini

1. Go to [aistudio.google.com](https://aistudio.google.com) and sign in with your Google account.
2. Click **Get API Key** in the sidebar → **Create API key**.
3. Choose or create a Google Cloud project, then copy the key.

The Gemini API has a free tier with no credit card required — generous rate limits for personal use.

---

## DeepSeek

1. Go to [platform.deepseek.com](https://platform.deepseek.com) and sign in.
2. Navigate to **API Keys** → **Create API Key**.
3. Copy the key. DeepSeek is one of the cheapest providers per token.

---

## xAI (Grok)

1. Go to [console.x.ai](https://console.x.ai) and log in with your X (Twitter) account.
2. Navigate to **API Keys** → **Create API Key**.
3. Copy the key.

---

## Mistral

1. Go to [console.mistral.ai](https://console.mistral.ai) and sign in.
2. Navigate to **API Keys** → **Create new key**.
3. Copy the key — it won't be shown again.

<Note>
  Codestral (Mistral's code model) may require a separate key. Visit [codestral.mistral.ai](https://codestral.mistral.ai) to request Codestral access.
</Note>

---

## Groq

1. Go to [console.groq.com](https://console.groq.com) and sign up.
2. Navigate to **API Keys** → **Create API Key**.
3. Copy the key.

Groq has a free tier with per-minute rate limits — great for personal use with no credit card needed.

---

## OpenRouter

1. Go to [openrouter.ai](https://openrouter.ai) and sign in.
2. Navigate to **Keys** → **Create Key**.
3. Optionally set a credit limit for the key.
4. Add credits under **Credits** — minimum top-up is usually \$5.

---

## Together AI

1. Go to [api.together.ai](https://api.together.ai) and sign up.
2. Your API key is shown on the **Settings** or **API Keys** page.
3. New accounts get \$1 in free credits.

---

## Fireworks AI

1. Go to [fireworks.ai](https://fireworks.ai) and sign up.
2. Navigate to **API Keys** → **Create API Key**.
3. New accounts get \$1 in free credits.

---
