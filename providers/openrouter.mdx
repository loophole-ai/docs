[OpenRouter](https://openrouter.ai) is a unified API that gives you access to models from Anthropic, OpenAI, Google, Meta, Mistral, and many others — all through one key and one billing account.

## Setup

<Steps>
  <Step title="Create an OpenRouter account">
    Sign up at [openrouter.ai](https://openrouter.ai) and add credits.
  </Step>
  <Step title="Generate an API key">
    Go to **Keys** in the OpenRouter dashboard and create a new key.
  </Step>
  <Step title="Enter the key in Loophole">
    Open **Settings** (`Ctrl+Shift+L`), select **OpenRouter**, paste your key, and save.
  </Step>
  <Step title="Add models">
    Loophole will load the model list from OpenRouter. Pick any model under **Settings → Model Selection**.
  </Step>
</Steps>

## Why use OpenRouter?

- **One key for everything** — no need to manage separate API keys for each provider.
- **Try models before committing** — experiment with Gemini, Claude, Llama, and others without signing up for each individually.
- **Fallback routing** — OpenRouter can automatically fall back to a backup provider if the primary is down.
- **Cost visibility** — see per-request costs across all models in one dashboard.
